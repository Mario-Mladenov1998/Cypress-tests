Cypress E2E Tests – SauceDemo

📌 Project Overview

This project contains End-to-End (E2E) automated tests for the SauceDemo web application using Cypress.

The goal is to validate core user flows such as login, cart navigation, and checkout functionality.

---

🚀 Technologies Used

- Cypress
- JavaScript
- Node.js

---

🧪 Test Scenarios

✅ Login Tests

- Valid login with correct credentials
- Invalid login with wrong username/password
- Error message validation

🛒 Cart Tests

- Open cart page
- Verify navigation to cart

💳 Checkout Tests

- Fill checkout information
- Complete order process

---

▶️ How to Run the Tests

1. Install dependencies:

npm install

2. Open Cypress:

npx cypress open
